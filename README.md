# 🎨 Etch-a-Sketch

A simple drawing app built using HTML, CSS, and JavaScript. Users can draw on a grid by hovering over cells.

## 🚀 Features
- Interactive grid
- Hover to draw
- Reset/Clear option

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🧠 Learning
- DOM manipulation
- Event handling
- Dynamic elements
